Protocol step

Activity

Clinical or operational action placed on the pathway map and reviewed by experts.

ECO-PATH module

LCA modeling zone assigned to pathway activities. It is not another clinical activity.

Resource flow

Measurable input or output inside a module, such as travel, equipment time, energy, materials, or waste.

LCI / LCIA link

Resource flows are mapped to inventory data and impact coefficients for calculation.

Modular LCA calculation demo

Editable calculation rows for candidate building blocks. Modules define the LCA boundary; row checkboxes define which building blocks are active in the current scenario. Each row uses a placeholder occurrence count and dummy impact coefficient so the interface can show the calculation logic and identify the data that still need to be collected.

Candidate building blocks by LCA module

Candidate ECO-PATH building blocks grouped by module. A selected pathway may use only part of a module, for example only the consultation and monitoring parts of EM06. EM10 and EM11 are optional cross-cutting support modules, not station modules shown on the pathway map.
